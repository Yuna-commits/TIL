## ADT
- 스택 생성 : Create()
- 스택이 비어있는지 검사 : is_empty()
- 스택이 가득 찼는지 검사 : is_full()
- 스택의 맨 위에 데이터 추가 : push(data)
- 스택의 맨 위 데이터 삭제 : pop()
- 스택의 맨 위 데이터 조회 : peek()
